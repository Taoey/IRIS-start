# IRIS-start
IRIS（go语言web框架）脚手架项目


### 热部署
使用：[gowatch](https://github.com/silenceper/gowatch)进行程序热部署

安装：`go get github.com/silenceper/gowatch`
安装完成之后，直接使用`gowatch`命令即可

- [ ] REST风格
- [ ] 模板引擎风格

- [ ] 文件上传
- [ ] csv文件处理
- [ ] 配置文件读取
